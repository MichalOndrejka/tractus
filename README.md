# Tractus

An autonomous "software company" of AI agents that pull work from a backlog and build it under your approval. Runs locally.

## Getting started

```bash
cp .env.example .env   # fill in GITHUB_TOKEN + GITHUB_REPOS
npm install
npm run dev            # backend + web
```

- Backend: http://localhost:8787
- Web: http://localhost:5173

See [`docs/ROADMAP.md`](docs/ROADMAP.md) for the full design and roadmap.
